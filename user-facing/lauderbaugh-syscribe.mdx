You can click through directly to review any pending drafts. ### Activity Feed Filtering The activity feed now includes filter tabs so you can quickly narrow down what you're looking at: - \*\*All\*\* — everything in your feed - \*\*Pending\*\* — drafts waiting for review - \*\*Deployed\*\* — drafts that have gone live - \*\*No Updates\*\* — pull requests that didn't require any documentation changes - \*\*Failed\*\* — drafts that encountered an error Use the tabs at the top of the activity list to switch between views.
