# Getting Started with Syscribe

Syscribe automatically keeps your documentation up to date whenever your code changes. Connect a GitHub repo, configure your documentation targets, and Syscribe handles the rest.

## Quick start

1.  Sign up and create an organisation at app.syscribe.dev
2.  Go to Settings → Repos and connect your GitHub repository
3.  Configure which documentation targets each doc type should push to
4.  Add the Syscribe webhook to your repo in GitHub Settings → Webhooks
5.  Merge a PR — Syscribe will generate drafts and notify you by email

## Supported targets

Syscribe can push documentation to Notion, Mintlify, GitHub Pages, Gitbook, and more. Each doc type in your repo can be routed to a different target.
